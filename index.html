The difference between padding, border and margin is:

  • margin - used to generate space around elements
  • padding - The padding property in CSS defines the innermost portion of the box model, creating space around an element's content, inside of any defined margins and/or borders.
  • border - The border property defines the space between an element's padding and margin.
